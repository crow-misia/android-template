Android Template
=================

Android app Template

**Features**

* Kotlin
* Kotlin Android Extensions
* Android Architecture Components (ViewModel, LiveData, Room ORM Database)
* Dagger
* RxJava2
* ThreeTenBp
* Retrofit
* Kotshi
* Kotpref
* PermissionDispatcher
* Stetho
* Debot
* Espresso Tests
* JUnit tests / mockito
* Timber
